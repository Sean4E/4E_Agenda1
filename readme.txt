# 4E Agenda Builder

A browser-based tool for creating, managing, and exporting workshop agendas.

![4E Agenda Builder Logo](4ELogo.png)

## Features

- Create multi-day workshop agendas with customizable sessions
- Drag and drop interface for easy session management
- Mark sessions as breaks or regular content
- Save and load agenda files for sharing or backup
- Export agendas to PDF or Excel format
- Canvas view for visual agenda planning
- Create reusable session templates
- Automatic calculation of timing and session counts
- Works on any device with a modern web browser
- Local storage saves your work automatically

## How to Use

1. **Add Sessions**: Use the form on the left to create new agenda items
2. **Organize**: Drag and drop to reorder sessions
3. **Multi-day Support**: Add multiple days for longer workshops
4. **Templates**: Save common sessions as reusable templates
5. **Export**: Generate PDF or Excel files for distribution
6. **Canvas View**: Toggle to visual mode for a different perspective

## Local Storage

The app uses your browser's local storage to automatically save your work. To work across multiple devices:

1. Use the "Save Agenda" button to download your agenda as a JSON file
2. Use the "Load Agenda" button to import previously saved files

## Getting Started

Simply open the web application in your browser. No installation or account needed.

## Compatibility

Works on all modern browsers including:
- Chrome
- Firefox
- Safari
- Edge
